<p> show me is an intelligent macOS mouse companion that understands your screen and can control your computer in-context. </p>

</div>

## What You Can Do

- Speak commands like:
  - "Open Notes and create a new note."
  - "help me learn how to use this app"
  - "Click Blank document."
  - "Open youtube.com."
  - "Press command shift 4."
    
- Highlight an area and issue local edit commands like:
  - "Rewrite this."
  - "Change this word."
  - "Move this over there."


## Controls

- **Ctrl + Option**: toggle live voice dictation session on or off.
- **Ctrl + Shift + drag**: paint a freeform focus highlight.
- **Menu bar icon**: open panel, key setup, permissions, and mode toggles.

## Modes

- **Autopilot** (default): ShowMe executes actions end to end.
- **Teaching**: ShowMe points and guides while you do the clicking



## Permissions

ShowMe requires:

| Permission | Purpose |
|---|---|
| Microphone | Voice input streaming |
| Screen Recording | Visual context screenshots |
| Accessibility | UI reading and computer control |

# Privacy

- We do not sell your data.
- We do not intentionally collect passwords, payment card numbers, two-factor codes, or secret tokens.
- showme is designed to avoid automatically typing into secure fields such as password fields.
